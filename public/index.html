Welcome to the API homepage for Northcoders News

This page will provide some useful information on how to navigate the API along with some of the endpoints available to you.
This API is designed to simulate a news website where clients are able to navigate between different articles, topics and users. Queries can be added to the end of endpoints to sort, filter and delete data as required.

Useful links:

  • Github Repo

Endpoints available to you:

/api/topics

  • Method: GET
  • Description: Returns an object containing an array with the information of all the topics
  • Queries: None
  • Example response: 
     { topics: [ { slug: 'football', description: 'Footie!' }, {...} ] }

/api/users

  • Method: GET
  • Description: Returns an object containing an array with all the information on each user
  • Queries: None
  • Example response: 
     {"users":[{"username":"tickle122","name":"Tom Tickle","avatar_url": url-link}, {...}, ... ]}

/api/articles

GET

  • Description: Returns an object with an array with information on every article
  • Queries: Yes
    • To sort an articles by a specific column in asc or desc: /api/articles?sort_by=:column&order=:order
    • To filter articles by a specific topic: /api/articles?topic=:topic
  • Example response: 
    {"articles":[{"article_id":34,"title":"The Notorious MSGâ€™s Unlikely Formula For Success","topic":"cooking","author":"grumpy19", 
    "created_at":"2020-11-22T11:13:00.000Z","votes":0,"article_img_url": url-link,"comment_count":11}, {...} ... ]}
  • Other article end points:
    • To view a specific article via article_id: /api/articles/:article_id
    • To view comments on an article via article_id: /api/articles/:article_id/comments

PATCH

  • Description: Allows you to update the votes on a specific article and returns the updated article
  • Path: /api/articles/:article_id
  • Requested body format: 
    { inc_votes: 100 }
    to increase votes by 100, or 
    { inc_votes: -50}
    to decrease votes by 50

POST

  • Description: Allows you to add a comment to a specified article and responds with the added comment
  • Path: /api/articles/:article_id/comments
  • Requested body format: 
    { username: "butter_bridge", body: "This is a test body." }
