Welcome to the API homepage for Northcoders News

This page will provide some useful information on how to navigate the API along with some of the endpoints available to you.
This API is designed to simulate a news website where clients are able to navigate between different articles, topics and users. Queries can be added to the end of endpoints to sort, filter and delete data as required.

Useful links:

  • Github Repo

Endpoints available to you:

/api/topics

  • Method: GET
  • Description: Returns an object containing an array of all topics
  • Queries: None
  • Example Response: 
     { topics: [ { slug: 'football', description: 'Footie!' }, {...} ] }
